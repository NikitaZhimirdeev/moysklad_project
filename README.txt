aquapolis.py - исполняющий модуль по работе с сайтом aquapolis;
Aquapolis_function.py - модуль в котором прописаны дополительные функции,
		которые необходимы для работоспособности aquapolis.py;
Aquapolis_data.json - файл в котором храняться все загруженне на МС товары;
LOG_aquapolis.txt - файл логов, логи хранятся в фомате ( -- <начало работы> -- <конец работы> -- <кол-во добавленных> -- <кол-во обнавленных> -- <кол-во перемещенных в архив>)

Astral.py - исполняющий модуль по работе с сайтом Astral;
astral_function.py - модуль в котором прописаны дополительные функции,
		которые необходимы для работоспособности Astral.py ;
Astral_data.json - файл в котором храняться все загруженне на МС товары с сайта Astral;
LOG_astral.txt - аналогично LOG_aquapolis.txt

email_mes.py - скрипт отправки сообщений на почту
config.py - модуль хранения настроек
